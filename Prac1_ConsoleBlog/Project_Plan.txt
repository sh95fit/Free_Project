# 프로젝트 기획 의도
'''
게시판과 유사한 콘솔형 블로그 제작

CRUD 기반의 프로그램 제작 연습
C(create, 쓰기), R(read, 읽기), U(update, 수정), D(delect, 삭제)
'''

# 프로젝트 설계
'''
프로젝트 설계를 왜 하는가?
-> 설계 없이 프로젝트를 진행하면 원하는 목적 달성과 멀어질 수 있음

진행 단계
요구사항 분석 -> 프로그램 설계(기능 구현 방안) -> 프로그램 개발 -> 테스트
'''

# 프로그램 설계
'''
프로그램 기능 파악하기
 1. 게시글 로딩하기
 2. 메뉴 출력하기
 3. 게시글 쓰기 (C)
 4. 게시글 목록 확인
 5. 게시글 상세 확인 (R)
 6. 게시글 수정 (U)
 7. 게시글 삭제 (D)
 8. 게시글 저장
'''

# 클래스 설계
'''
공통 데이터 - 게시글

게시글은 어떤 데이터를 가지고 있을까?
글번호, 제목, 본문 내용, 조회수

해당 내용을 클래스로 구현하면 코드 간소화에 유리

-> 클래스로 만들 데이터 - 게시글
-> 클래스 속성 : 글번호, 제목, 본문 내용, 조회수
-> 클래스 메서드 : 게시물 수정, 조회수 증가, 속성 가져오기(메서드를 통해 가져오는 것 - 게시글 목록, 상세 확인용)
'''
